object.is()，==和===

  • 使用双等号（==）进行相等判断时，如果两边的类型不一致，则会进行强制类型转化（布尔值，数值，原始值）后再进行比较。
  • 使用三等号（===）进行相等判断时，如果两边的类型不一致时，不会做强制类型准换，直接返回 false。
  • 使用 Object.is 来进行相等判断时，一般情况下和三等号的判断相同，它处理了一些特殊的情况，比如-0和+0不再相等，两个NaN是相等的。
//==相当于约等于
true == 1;//true，转为布尔值

'55' == 55;//true，转为数值

&#123;valueOf:function()&#123;return 1&#125;&#125; == 1;//true，转为原始值

null == undefined;//true

NaN == NaN;//false

let obj1 = &#123;name:"xxx"&#125;
let obj2 = &#123;name:"xxx"&#125;
let result1 = (obj1 == obj2 ); // false

//常见练习：操作符两边的值都尽量转成number
'' == '0' // false，类型相同，不存在类型转换
0 == '' // true，转为数值
0 == '0' // true，转为数值

false == 'false' // false
false == '0' // true
false == undefined // false
false == null // false
null == undefined // true

' \t\r\n' == 0 // true
